## 웹사이트

[링크](https://shareground.vercel.app/)

## 소개

ShareGround는 블로그 플랫폼으로서 글을 작성해서 다른 사람과 공유할 수 있습니다.

주요 기능은 다음과 같습니다.

- 파일 업로드 및 다운로드

- 댓글 작성

- 관심 글 추가

- 어떠한 글을 작성했는지 확인

- 글 검색
